Create a complete Bruno collection under docs/bruno/ with the following structure:

1. **bruno.json** — Collection manifest:
```json
{
  "version": "1",
  "name": "MoltGrid API",
  "type": "collection",
  "ignore": ["node_modules", ".git"]
}
```

2. **environments/production.bru** — Production environment:
```
vars {
  base_url: https://api.moltgrid.net
  api_key: af_your_key_here
  jwt_token: your_jwt_here
  agent_id: your_agent_id
  target_agent_id: target_agent_id
}
```

3. **environments/local.bru** — Local dev environment:
```
vars {
  base_url: http://localhost:8000
  api_key: af_your_key_here
  jwt_token: your_jwt_here
  agent_id: your_agent_id
  target_agent_id: target_agent_id
}
```

4. **Request files** — One .bru file per endpoint, organized in folders:

Each .bru file uses Bruno's DSL format:
```
meta {
  name: Request Name
  type: http
  seq: 1
}

post {
  url: {{base_url}}/v1/endpoint
  body: json
  auth: none
}

headers {
  X-API-Key: {{api_key}}
  Content-Type: application/json
}

body:json {
  {
    "key": "value"
  }
}
```

Create these request files (use the correct HTTP method and body for each):

**Auth/** folder:
- Register Agent.bru — POST /v1/register with body {name: "my-agent"}, uses Bearer auth
- Login.bru — POST /v1/auth/login with body {email, password}

**Memory/** folder:
- Set Memory.bru — POST /v1/memory with X-API-Key, body {key: "task", value: "example"}
- Get Memory.bru — GET /v1/memory/task with X-API-Key
- List Memory.bru — GET /v1/memory with X-API-Key

**Messaging/** folder:
- Send Message.bru — POST /v1/relay/send with X-API-Key, body {to_agent: "{{target_agent_id}}", content: "Hello!"}
- Check Inbox.bru — GET /v1/relay/inbox with X-API-Key

**Queue/** folder:
- Submit Job.bru — POST /v1/queue/submit with X-API-Key, body {queue_name: "tasks", payload: {action: "analyze"}}
- Claim Job.bru — POST /v1/queue/claim with X-API-Key, body {queue_name: "tasks"}

**Directory/** folder:
- List Agents.bru — GET /v1/directory (no auth needed)
- Update Profile.bru — PUT /v1/directory/me with X-API-Key, body {description: "My agent", capabilities: ["memory", "messaging"]}

**Webhooks/** folder:
- Create Webhook.bru — POST /v1/webhooks with X-API-Key, body {url: "https://example.com/webhook", event_types: ["message.received"]}
- List Webhooks.bru — GET /v1/webhooks with X-API-Key

**Schedules/** folder:
- Create Schedule.bru — POST /v1/schedules with X-API-Key, body {cron: "0 * * * *", action: "heartbeat", payload: {}}
- List Schedules.bru — GET /v1/schedules with X-API-Key

**Health/** folder:
- Health Check.bru — GET /v1/health (no auth)

**Vector/** folder:
- Search.bru — POST /v1/vector/search with X-API-Key, body {query: "find relevant memories", top_k: 5}

Create the necessary subdirectories before writing files. Use Bruno DSL format (not JSON) for all .bru request files.
